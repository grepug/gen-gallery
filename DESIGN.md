# Design

## Visual Theme

- Quiet internal tool
- Light workspace for image review
- Restrained color strategy with one cool accent

## Colors

- App background: soft cool neutral
- Panel surfaces: slightly brighter neutral layers
- Primary accent: restrained blue for active state, selection, and primary actions
- Semantic colors: amber for waiting, green for success, red for failure

## Typography

- System sans stack
- Tight product scale
- Strong contrast between page title, panel labels, and metadata text

## Layout

- Main workspace split into gallery area and persistent detail pane
- Masonry-like thumbnail grid for browsing
- Sticky detail pane with large preview, vertical nearby-thumbnail rail, and actions

## Components

- Filter chips
- Status pills
- Thumbnail cards with image and prompt snippet
- Detail pane with preview, metadata, retry/delete actions
- Empty and loading states that feel native to the tool

## Motion

- Minimal transitions only for selection and hover feedback
- No decorative page choreography
